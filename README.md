<h1>Movie Recommendation Capstone Project</h1>

<h2>Description</h2>
<p>This is a Python based movie recommendation app for my capstone project at WGU. The app is based on the "TMDB 5000 Movie Dataset" found at https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata. The app uses a machine learning model based on a content-based filtering method to generate movie recommendations. The app was developed in PyCharm and requires the two .csv dataset files from the link above</p>

<h2>User Guide</h2>

<p>Below is a brief user guide to start using the app through PyCharm</p>
<ol>
  <li>Install Python 3.12 or later</li>
  <li>Install PyCharm Community Edition</li>
  <li>Install the required Python libraries using the following command in the PyCharm terminal<br>pip install pandas numpy matplotlib scikit-learn</li>
  <li>Ensure the two dataset files are in the directory</li>
  <li>Open the project in PyCharm</li>
  <li>Run the main.py file</li>
  <li>Once the output displays the message "Switch to Movie Recommendation Window", switch to the Tkinter GUI</li>
  <li>Enter the movie title and click on the Get Recommendations Button</li>
  <li>Review the list of recommended movies and similarity percentages displayed in the results area</li>
  <li>Use the visualization buttons to explore movie popularity, genre, and ratings trends</li>
</ol>
